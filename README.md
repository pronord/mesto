# **GitHub Explorer**
Версия 0.0.2

## О проекте:
Проектная работа для демонстрации настройки webpack

## Основной функционал: 
- Можно изменить информацию о себе, для этого нужно нажать на кнопку "edit"
- Можно добавить свою картинку, для этого необходимо нажать на кнопку "+"
- Можно ставить лайк и удалять карточки
- Все карточки можно увеличить

## Стэк технологий:
ES6, Babel, OOP, CSS3, HTML5, Webpack, BEM

## Инструкции по запуску:
- Скачать или склонировать репозитори
- Установить зависимости при помощи npm - `npm i`
- Запустить в development режиме - `npm run dev`
- Запустить сборку production-билда - `npm run build`
- Разместить production-билд на github pages - `npm run deploy`
